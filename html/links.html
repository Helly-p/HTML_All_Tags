Links - Hyperlinks

  • Click on a link and jump to another document.
  • Example : Visit Website 
    <a href="url" > Click On me </a>

Target Attribute

  • The target attribute specifies where to open the linked document.
    • _self - Default. Opens the document in the same window/tab as it was clicked
      Example : Click Me
    • _blank - Opens the document in a new window or tab
      Example : Click Me
    • _parent - Opens the document in the parent frame
      Example : Click Me
    • _top - Opens the document in the full body of the window
      Example : Click Me

Absoulte URL vs Relative URL

Example of Absoulte URL : <a href="http://www.google.com/my-link.html">Click Me<a>


Example of Relative URL : <a href="./images/index.html">Click Me<a>
