Lists

Unordered List

Example : 
                <ul>
                    <li>Tea</li>
                    <li>Coffee</li>
                </ul>
Output :
  • Tea
  • Coffee

Ordered List

Example : 
                <ol type="A">
                    <li>Tea</li>
                    <li>Coffee</li>
                </ol>
Output :
  1. Tea
  2. Coffee

Description List

Example : 
                <dl>
                    <dt>Coffee</dt>
                    <dd>- black hot drink</dd>
                    <dt>Milk</dt>
                    <dd>- white cold drink</dd>
                  </dl>
Output :
Coffee
- black hot drink
Milk
- white cold drink
