How webpage renders

  • First it will build DOM Tree using HTML elements
  • Fetch External Resources : CSS & JS
    • CSS : Block rendering : It will overwrite properties at various points, So it will stop rendering till the CSSOM not created.
    • JS : Block Parsing (Division of elements) : It will stop the DOM creation, So it will use async or defer attribute, So it is preferable to put JS end of body tag.
  • Parse CSS & Create CSSOM
  • Execute JS
  • Merge DOM & CSSOM : Create Render Tree
  • Display Render Tree on Web browser
